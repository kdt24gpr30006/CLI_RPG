DungeonSceneとBattleSceneはゲームシーンとして統一する
それでゲーム内の共通データの管理を楽にする
共通データはGameDataで管理する
GameDataはプレイヤー情報(ステータス)、現在の階層、所持アイテム
マップデータについてDungeonSceneで管理する
マップデータはCSVなどで外部から読み込む形にして変更を楽にしときたい

バトルターンの管理の仕方
ターンはプレイヤーとエネミーを分ける
enum class BattleState { playerTurn, EnemyTurn, Win, Lose};

バトルマネージャーで管理する
class BattleManager {
    プレイヤーとエネミーを持つ
    Player* player;
    Enemy* enemy;
    BattleState state;

public:
    void Init(Player* p, Enemy* e);
    
    コマンド
    void ExecuteCommand_Attack();
    void ExecuteCommand_Magic();
    void ExecuteCommand_Item();
    
    void Update(); 敵のターン処理など
    void Render(); 戦闘画面の描画
};
